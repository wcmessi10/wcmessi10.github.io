# OCR(Optical Character Recognition) 정리

## 1. OCR이란?

**OCR(광학 문자 인식)**은 이미지 속의 텍스트를 **기계가 읽을 수 있는 디지털 텍스트 데이터로 변환**하는 기술입니다.
예) 스캔한 문서, 영수증, 계약서 이미지 → **편집/검색 가능한 텍스트**

---

## 2. OCR이 중요한 이유

### 기존 문제점

* 종이 문서 → 저장/관리 비용 증가
* 스캔 → 단순 이미지 → **검색, 편집 불가능**
* 수작업 데이터 입력 → **느리고 오류 많음**

### OCR 도입 효과

* 문서 **자동 디지털화**
* **검색, 분석, 자동화 가능**
* **업무 효율성 + 생산성 향상**
* 종이 없는 **Paperless Workflow 구축**

---

## 3. OCR 작동 방식 (처리 단계)

> **이미지 획득 → 전처리 → 텍스트 인식 → 패턴 매칭 → 특징 추출 → 후처리**

### 이미지 획득

* 스캐너, 카메라로 문서 스캔 → 이미지 생성

### 전처리

* 기울기 보정
* 노이즈 제거
* 선/텍스트 영역 분리
* 다국어 문자 인식 준비

### 텍스트 인식

* **패턴 매칭**
* **특징 추출**

### 패턴 매칭

* 글자 이미지를 기존 템플릿과 비교
* 단점: **새로운 폰트/필기체 인식 어려움**

### 특징 추출

* 선, 곡선, 교차점 등 **형태적 특징 분석**
* 머신러닝 기반 → **높은 정확도**

### 후처리

* 결과를 TXT, Word, PDF 등 문서로 변환

---

## OCR 기술 유형

| 유형                  | 설명              |
| ------------------- | --------------- |
| **단순 OCR**          | 템플릿 기반 문자 비교    |
| **ICR (지능형 문자 인식)** | 머신러닝 기반 문자 인식   |
| **IWR (지능형 단어 인식)** | 단어 단위 인식        |
| **OMR (광학 마크 인식)**  | 체크박스, 로고, 기호 인식 |

---

## 5. OCR 기술별 특징 비교

### 단순 OCR

* 패턴 매칭 방식
* 장점: 빠름
* 단점: **폰트 다양성 대응 불가**

### 지능형 문자 인식(ICR)

* 딥러닝, 신경망 기반
* 필기체, 다양한 폰트 인식 가능
* **가장 널리 사용**

### 지능형 단어 인식(IWR)

* 문자 분해 없이 **단어 단위 인식**
* 필기체 문서에서 강점

### 광학 마크 인식(OMR)

* 로고, 기호, 워터마크 탐지

---

## 6. OCR 기술의 주요 이점

### 검색 가능

* 문서 검색
* 빅데이터 분석
* 자동 분류

### AI 활용

* 차량 번호판 인식
* 로고 감지
* 광고 이미지 분석

### 운영 효율 향상

* 문서 자동 처리
* 업무 자동화
* 비용 절감

---

## 7. OCR 활용 예시

### 문서 자동화

* 영수증 자동 입력
* 계약서 자동 등록
* 보험 청구 자동 처리

### 데이터 검색

* 문서 자동 분류
* 키워드 기반 빠른 검색

### 필기 텍스트 변환

* 손글씨 → 디지털 문서
